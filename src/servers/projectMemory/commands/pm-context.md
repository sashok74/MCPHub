# /pm-context — Сбор контекста перед задачей

Запускает агент `pm-context` через Task tool для сбора контекста из project-memory и dbmcp.

## Использование

```
/pm-context <описание задачи>
```

## Инструкция

Запусти агент pm-context через Task tool:

```
Task(subagent_type="pm-context", prompt="<описание задачи от пользователя>")
```

Передай в prompt полный текст задачи пользователя без изменений. Агент сам проанализирует промпт, соберёт контекст и вернёт структурированный бриф. После получения брифа — выведи его пользователю.
